# Changelog

Tüm önemli değişiklikler bu dosyada belgelenecektir.

## [1.0.0] - 2026-05-20

### ✨ Eklenenler
- İlk sürüm yayınlandı
- Modüler CSS ve JavaScript yapısı
- Dark mode desteği (sistem tercihine göre otomatik)
- Klavye navigasyonu (Tab, Enter, Arrow keys)
- IntersectionObserver ile scroll animasyonları
- ARIA labels ve erişilebilirlik iyileştirmeleri
- Vite build sistemi
- Responsive design (mobil, tablet, desktop)
- Glassmorphism UI tasarımı
- 5 adımlı staj süreci timeline'ı

### 🎨 Tasarım
- SDU kurumsal renkleri (Lacivert #143264, Kırmızı #b4141e)
- Inter font ailesi
- Phosphor Icons entegrasyonu
- Smooth cubic-bezier animasyonlar

### ♿ Erişilebilirlik
- WCAG 2.1 AA standartlarına uyum
- Semantic HTML5 yapısı
- Focus management
- Screen reader desteği
- `prefers-reduced-motion` desteği

### ⚡ Performans
- Font preconnect
- CSS modülleri
- ESM JavaScript modülleri
- Lazy-load hazır altyapı
